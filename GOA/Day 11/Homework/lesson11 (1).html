HTML

HTML არის ყველა ვებ გვერდის ბირთვი, მიუხედავად საიტის სირთულისა, ეს არის აუცილებელი უნარი ნებისმიერი ვებ პროფესიონალისთვის. ეს არის საწყისი წერტილი მათთვის, ვინც სწავლობს როგორ შექმნას კონტენტი ვებისთვის. და, ჩვენდა საბედნიეროდ, საოცრად ადვილია სწავლა.

CSS

CSS გამოიყენება პრეზენტაციის, ფორმატირებისა და განლაგების გასაკონტროლებლად. ეს პროგრამირების ენა კარნახობს, თუ როგორ უნდა გამოჩნდეს ვებსაიტის HTML ელემენტები გვერდის წინა მხარეს.

JS

JavaScript უფრო რთული ენაა, ვიდრე HTML ან CSS. დღესდღეობით, JavaScript მხარდაჭერილია ყველა თანამედროვე ვებ ბრაუზერის მიერ და გამოიყენება თითქმის ყველა საიტზე ინტერნეტში უფრო ძლიერი და რთული ფუნქციონირებისთვის.
